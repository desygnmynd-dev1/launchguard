# SaaS Starter Project# SaaS Starter Project

This is the base scaffold used for all SaaS builds.

Do NOT modify architecture casually.
Improvements should be pushed back to saas-execution-system repo.

---

## Stack

- React
- Vite
- Multi-tenant ready data structure
- Role-based architecture
- Risk engine scaffold
- Netlify deployment ready

---

## Branch Structure

main → production  
dev → integration  
feature/* → new work  

---

## Setup

1. npm install
2. npm run dev
3. npm run build

---

## Deployment

Connected to Netlify.
Auto-deploy from main.
Preview from dev.

---

## Architecture Principles

- Multi-tenant aware
- Deterministic logic
- No hidden state mutations
- No console errors
- Clear separation of concerns
